# Project Specifications

Define your tasks here. The AI will work through them one at a time.

## Example Format (Checkbox)

- [ ] First task to complete
- [ ] Second task to complete
- [ ] Third task to complete

## Alternative Formats

You can use any format you prefer:
- TODO/DONE markers
- Numbered lists with status
- Sections for different features

The AI reads this file each iteration and determines what to work on next.
